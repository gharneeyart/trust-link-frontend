# Security Policy

## Supported Versions

This project currently supports the latest main branch and the most recent release line.

## Reporting a Vulnerability

Please report security issues privately by contacting the maintainers through the repository's security contact channel.

## Security Practices

- Keep dependencies up to date.
- Use secure environment variables for wallet and API configuration.
- Avoid exposing private keys or secrets in code, logs, or issues.
- Review third-party integrations and wallet interactions carefully.

## Response Expectations

We aim to acknowledge valid reports promptly and evaluate fixes as quickly as possible.
